# Sales_Performance_Analysis_Dashboard
Built an interactive Sales Performance Dashboard in Excel to analyze revenue trends, product performance, and regional sales. Used Pivot Tables, Charts, KPI Cards, and Slicers to generate business insights from 300 sales records.

# Sales Performance Analysis Dashboard (2023–2025)

## Project Overview

This project analyzes sales data from 2023 to 2025 using Microsoft Excel.

The dashboard provides insights into:

* Sales Trends
* Product Performance
* Category Analysis
* City-wise Sales
* Region-wise Sales
* Payment Mode Analysis
* Salesperson Performance

## Tools Used
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards

## Dashboard Features
* Interactive Slicers
* Revenue Analysis
* Product Performance Tracking
* Regional Sales Analysis
* Salesperson Performance Evaluation

## Dataset Information
The dataset contains 300 sales records from 2023 to 2025.
